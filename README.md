## Como executar o projeto

Entrar na pasta da api (node-api)
```bash: 
cd /score-api
```
instalar dependencias
```bash: 
npm install
```

Executar api
```bash: 
npm run start:dev
```

Executar testes
```bash: 
npm run start:test
```
___

Entrar na pasta da aplicação react 
```bash: 
cd /score-app
```

Instalar dependencias
```bash: 
npm install
```
Executar aplicação
```bash: 
npm run dev
```
